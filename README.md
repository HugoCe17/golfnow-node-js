Node.js SDK for GolfNow
===============

Node.js SDK for GolfNow

## Installation

* Install [Node.js](https://nodejs.org/)
* Install Api.ai SDK with `npm`:

`npm install apiai`


## Usage

    var numFormatter = require('@jdaudier/number-formatter');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`


## Tests

  `npm test`
